\# ✈️ GreenWing Analytics



Havayolları için veri odaklı yakıt verimliliği ve emisyon analitiği.



\## 🎯 Özellikler



\- \*\*Yakıt Verimlilik Analizi\*\* - Filo genelinde yakıt tüketimi takibi

\- \*\*CORSIA Uyumluluk\*\* - Otomatik emisyon raporlaması

\- \*\*İnteraktif Dashboard\*\* - Gerçek zamanlı KPI takibi

\- \*\*Otomatik PDF Raporlar\*\* - Profesyonel, yönetim kuruluna sunulabilir raporlar
